This guide assumes you have the following basic background:

Hello world

To get started with writing Python, install python, open the editor of your choice and write:

print("hello world")

After that save the code in a file with a .py extension and run it in your editor or your console using python filename.py

Variables

You use variables as symbolic names for values in your application. The names of variables, called identifiers, conform to certain rules.

A Python identifier must start with a letter, underscore (_), or dollar sign ($); subsequent characters can also be digits (0-9). Because Python is case sensitive, letters include the characters "A" through "Z" (uppercase) and the characters "a" through "z" (lowercase).

You can use unicode characters as well, but only the ones that resemble letter characters.

Declaring variables

Unlike some other languages, declaring variables in python is extremely simple.

All you need to do is give it a name and a value!

x = 24

name = "John"

truth = True

Variables in python don't need specific type notation, they can be numbers, strings, truth values etc.

The syntax above declares a block scope local variable. See Variable scope below.

Variable scope

When you declare a variable outside of any function, it is called a global variable, because it is available to any other code in the current document. When you declare a variable within a function, it is called a local variable, because it is only available inside that function.

def add_numbers(a,b):

    return a + b

The example given above is a function declaration. Don't worry, those will be covered a little bit later. We can see the function has 2 variables, a and b. These variables only exist inside of the function and if we try and use them anywhere else we will get an error!

Global variables

Global variables are all variables declared inside of the main program and not inside of any function. They can be accessed by and changed from any other function we write in the program.

Data types

Python has primitive and complex data types. The primitive types are:

  • String - strings of text like "John", wrapped in double or single quotes
  • Numbers - int for integers or whole numbers and float for decimal numbers
  • Boolean - a truth value, it can be either True or False
  • None - a value that signifies nothing

Python also contains some more complex types like:

  • Sequence types - list, tuple, range
  • Mapping type - map
  • Set types - set, frozenset
  • Binary types - bytes, bytearray, memoryview

Alongside those, OOP - object-oriented programming can also be done in Python since it supports classes. More on them later.

if...else statements

Use the if statement to execute a statement if a logical condition is true. Use the optional else clause to execute a statement if the condition is false. An if statement looks like this:

if condition_1:

    statement_1

elif condition_n:

    statement_n

else:

    statement_last

If a condition is true, the statement inside of it gets executed and all other statements are skipped and ignored! If none of the conditions are true and there is no else clause, everything gets skipped.

While statements

A while statement is continuously executed while the specified condition is true. This is what it looks like:

while condition:

    statement

Following that logic you could write an infinite while loop as follows:

while True:

    statement

Function declarations

Function declarations in Python look really simple. It's the name of the function, followed by the parameters it accepts, followed by a double colon and then the function definition indented by 1 tab (4 spaces usually). e.g:

def sum(a,b):

    return a + b

In the example above, the name of the function is sum, it accepts 2 parameters, a and b. The return keyword tells the function to return a number with the value of a + b to the place it was called.

c = sum(3+5)

If we then call the function and save it's return value into a variable c, we will get a variable c with the value 8!

Classes

Python is an object-oriented language. Almost everything in python is an object, with its properties and methods. A Class is like an object constructor, or a "blueprint" for creating objects. To create a class you would use the keyword class:

class MyClass:

    properties and methods

We can then create an instance of that class:

my_object = MyClass()

Classes are a more complex feature of python and outside of this guides scope. If you want more information checkout Classes
